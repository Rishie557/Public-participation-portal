# Sauti ya Wananchi

A citizen-driven platform for democratic tax policy, transparency, and accountability in Kenya.

## Files

- `index.html` — main webpage
- `style.css` — site styling
- `script.js` — interactive behavior
- `README.md` — project description

## How to run

1. Open the folder in VS Code.
2. Open `index.html` in your browser.
3. Make sure `style.css` and `script.js` are in the same folder.

## GitHub Pages

After pushing to GitHub, enable GitHub Pages from:

**Settings → Pages → Branch: main → /(root)**
